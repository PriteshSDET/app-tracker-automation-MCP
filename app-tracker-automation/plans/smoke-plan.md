# Smoke Test Plan

## Purpose
Validate core business functionality and critical user journeys.

## Test Scope
- User authentication flows
- Main business processes
- Key feature functionality
- Integration points

## Execution Criteria
- Run on code merges
- Maximum 15 minutes execution time
- Critical path validation
- Cross-browser compatibility

## Test Cases
1. Complete user registration flow
2. Policy quote generation
3. Claim submission process
4. Payment processing
5. Document upload functionality
6. Email notifications
7. Search and filtering
8. Report generation

## Success Metrics
- 95% pass rate minimum
- < 2 minutes average test time
- Critical path validation complete

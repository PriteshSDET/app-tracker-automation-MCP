# Test Healing Rules Prompt

## Purpose
Define auto-healing strategies for test failures.

## Healing Strategies
1. **Selector Healing**
   - Try alternative selectors
   - Use relative positioning
   - Implement dynamic waiting

2. **Timing Healing**
   - Adjust wait timeouts
   - Add retry mechanisms
   - Implement smart delays

3. **Element Healing**
   - Find similar elements
   - Use text-based locators
   - Implement fallback strategies

## Healing Triggers
- Element not found
- Timeout errors
- Stale element references
- Click intercepts

## Healing Limits
- Maximum retry attempts
- Timeout thresholds
- Fallback strategies
- Manual intervention points

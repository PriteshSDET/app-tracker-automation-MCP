# Bug Reporting Prompt

## Purpose
Generate comprehensive bug reports from test failures.

## Bug Report Template
- Bug ID and title
- Severity and priority
- Environment details
- Reproduction steps
- Expected vs actual results
- Screenshots/videos
- Error logs
- Root cause analysis
- Suggested fixes

## Classification
- Critical: Blocker issues
- High: Major functionality broken
- Medium: Minor issues
- Low: Cosmetic problems

## Auto-healing Triggers
- Element selector changes
- Page layout modifications
- Timing issues
- Network delays

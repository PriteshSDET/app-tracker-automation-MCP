# Test Case Generation Prompt

## Purpose
Generate comprehensive test cases from analyzed stories.

## Input Format
- Analyzed story output
- Test scenarios list
- Business requirements
- Technical specifications

## Generation Guidelines
1. Create positive and negative test cases
2. Include boundary value testing
3. Add error handling scenarios
4. Define test data requirements
5. Specify expected results

## Output Format
- Test case ID and title
- Preconditions
- Test steps
- Expected results
- Priority and tags
- Test data requirements
